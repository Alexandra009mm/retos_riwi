# 📦 Project Base README

## 📖 Description (English)
This project is a console-based system designed to manage and process data efficiently. It allows users to interact through a structured menu, perform different operations, and handle information in an organized way. The system includes input validation to ensure data accuracy and prevent errors during execution.

Additionally, it provides a summary of the processed data, enabling users to clearly analyze results. This project demonstrates fundamental programming concepts such as modular design, data handling, and user interaction. It can be easily adapted to different scenarios such as stores, hospitals, banks, or reservation systems.

---

## 📖 Descripción (Español)
Este proyecto es un sistema basado en consola diseñado para gestionar y procesar datos de manera eficiente. Permite a los usuarios interactuar a través de un menú estructurado, realizar diferentes operaciones y manejar la información de forma organizada. El sistema incluye validación de datos para asegurar la precisión y evitar errores durante la ejecución.

Además, proporciona un resumen de los datos procesados, permitiendo analizar los resultados de manera clara. Este proyecto demuestra conceptos fundamentales de programación como diseño modular, manejo de datos e interacción con el usuario. Puede adaptarse fácilmente a diferentes escenarios como tiendas, hospitales, bancos o sistemas de reservas.
